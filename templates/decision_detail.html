How to verify independently

1. Fetch the pure-commitment envelope from Arweave:

curl https://turbo-gateway.com/raw/{{ envelope.arweave_tx_id }}

2. Verify the Ed25519 signature using the embedded public_key. The signed body is the envelope minus the signature field, JCS-canonicalized (RFC 8785).

3. Download the canonical bytes from MLflow:

mlflow artifacts download -r <run_id> -a ario/predictions/<decision_id>/payload.json

Compute SHA-256 of those raw bytes and compare to the envelope's payload_hash.

4. Verify previous_hash matches the model version's registration TX (chain integrity).

Or run ario-mlflow verify trace {{ envelope.record.trace_id }} for all four checks at once: signature, anchored-bytes intact, source-of-truth (re-fetches the trace's ario.payload_json tag and compares to the artifact), and ar.io attestation.
